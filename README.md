# 🏡 ConnectingHostels

A full-stack real-time listing platform tailored for hostel seekers and owners in localities like Mangalpally, Hyderabad.  
*ConnectingHostels* bridges the gap between hostel owners and students by providing verified, filterable listings based on gender, floor, co-living types, and more.

> Built to simplify student life. Powered by modern web tech. Inspired by real hostel needs.

---

## 🚀 Features

- 🔐 Hostel Owner & Student Authentication
- 🏠 Add, View & Manage Hostel Listings
- 📷 Upload Hostel Images with Room Info
- 🧭 Filter by Gender (Boys/Girls/Co-live), Floor, Type
- 📍 Location-based Listings (Manual Input & GPS)
- 🌙 Global Dark Mode Support
- 🖥️ Responsive Design (Mobile + Web Friendly)

---

## 🛠 Tech Stack

- **Frontend**: React.js, Tailwind CSS  
- **Backend**: Node.js, Express.js, MongoDB, JWT Auth  
- **Maps**: Google Maps API (for location features)  
- **Hosting**: Netlify (Frontend), Render (Backend)
